## One Line
Codebolt is a high-performance IDE precision-engineered for AI agents at scale.

## One Paragraph
Codebolt is a ground-up IDE built for the era where AI agents don't just
assist development — they perform it. While most tools optimize for better
models or smarter prompts, Codebolt solves the harder problem: making agents
coordinate, resolve conflicts, and scale reliably across codebases and
environments. The result is a precision-engineered environment where multi-agent
workflows actually work. Designed for engineers who treat AI as infrastructure,
not just assistance.

## Extended Description
Every AI development tool today operates on the same assumption: make the
agent smarter and the output gets better. Codebolt starts from a different
premise — that the environment agents operate in matters more than the agents
themselves.

A brilliant engineer dropped into a company with no version control, no CI/CD,
and no communication tools would be ineffective. Not because they lack skill,
but because the environment doesn't support good work. The same is true for
AI agents today. They operate in environments that give them no memory, no
coordination primitives, no conflict resolution, and no drift tracking.
So they fail. And everyone blames the model.

Codebolt is the environment that changes this. Built from the ground up — not
a VS Code fork, not a wrapper — it provides multi-agent coordination,
cross-environment reconciliation, drift tracking, and conflict resolution as
first-class system primitives. Agents in Codebolt work as coordinated teams,
not independent processes fighting over shared state.

The architecture draws on distributed systems research including
stigmergy-based coordination, narrative versioning, and structured execution
graphs. These aren't bolted-on features — they're the foundation the IDE
is built on top of.

## Origin
Codebolt was created because we saw the gap between what AI agents promise
and what they can actually deliver in production. The bottleneck isn't
intelligence. It's infrastructure. We built the infrastructure.

## What Codebolt Is Not
- Not a VS Code fork with AI features added
- Not a copilot or code completion tool
- Not an autonomous "AI employee" that works without direction
- Not a prompt engineering platform
- Not a general-purpose agent orchestration framework
