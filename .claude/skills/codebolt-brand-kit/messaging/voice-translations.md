## Product Copy

**Do:** "Agents coordinate across environments. Conflicts resolve at the system level."
**Don't:** "Our amazing agents seamlessly coordinate across all your environments!"

**Do:** "Ground-up architecture for multi-agent execution."
**Don't:** "We've revolutionized the IDE from the ground up with next-gen AI capabilities."

**Do:** "Drift tracking catches misalignment before it compounds."
**Don't:** "Never worry about agent drift again with our powerful drift detection engine!"

## Marketing / Landing Page

**Do:** "The High-Performance Environment for AI Agents."
**Don't:** "The World's Most Advanced AI Development Platform."

**Do:** "Scaling agents is an engineering problem. We solved the engineering."
**Don't:** "We're disrupting the AI development space with our revolutionary approach!"

**Do:** "Designed for engineers who treat AI as infrastructure — not just assistance."
**Don't:** "Built for everyone who wants to supercharge their AI coding workflow!"

## Social Media

**Do:** "Most agent failures aren't model failures. They're environment failures. That's what we fix."
**Don't:** "Excited to announce our AMAZING new feature! 🚀🔥 Game-changer alert!"

**Do:** "New: cross-environment conflict resolution now handles 3-way agent merges. Details in docs."
**Don't:** "OMG we just shipped something incredible!! You're going to LOVE this! 🎉"

**Do:** Share architecture diagrams, benchmarks, technical deep-dives.
**Don't:** Share vague hype posts, feature lists without context, or "coming soon" teasers.

## Technical Documentation

**Do:** "The drift engine evaluates structural, scope, constraint, and coverage drift at two levels: local assessment within Rust engine instances and global coordination in JavaScript."
**Don't:** "Our super powerful drift engine uses cutting-edge technology to make sure your agents stay on track!"

**Do:** State what a feature does, when to use it, and what tradeoffs exist.
**Don't:** Describe every feature as essential, powerful, or best-in-class.

## Competitive Positioning

**Do:** "Most environments treat agents as independent processes. Codebolt treats them as a coordinated system."
**Don't:** "Unlike [competitor], which has terrible agent coordination, Codebolt is way better."

**Do:** "Traditional IDEs bolt AI onto an editor. Codebolt builds the editor onto an agent coordination layer."
**Don't:** "We're 10x better than Cursor/Copilot/Claude Code."

## Handling Limitations

**Do:** "Codebolt is built for engineers working with AI agents at scale. If you need a general-purpose editor, this isn't it."
**Don't:** "Codebolt is perfect for EVERYONE and EVERY use case!"

**Do:** "The system is in active development. Some edges are rough. The architecture is sound."
**Don't:** Pretend the product is flawless, or apologize excessively for being early-stage.

## Key Vocabulary

| Use This              | Not This                          |
|-----------------------|-----------------------------------|
| engineered            | revolutionary                     |
| precision             | magic / automagic                 |
| coordinates           | seamlessly orchestrates           |
| resolves              | eliminates forever                |
| handles               | supercharges                      |
| built for             | designed to delight               |
| scales                | unlocks infinite potential         |
| environment           | ecosystem (unless literal)        |
| ground-up             | from scratch (too casual)         |
| conflict resolution   | smart conflict handling           |
| drift tracking        | intelligent drift detection       |
| agents align          | agents harmonize                  |
