Memory Architecture

Persistent Memory

AI that remembers everything, forever. Knowledge graphs, vector databases, and episodic memory give agents permanent, queryable memory spanning your project history.

LLMs have limited context windows—100-200K tokens. That's a keyhole view of your codebase. Codebolt solves this at the IDE/harness level. Memory lives outside the agent, giving every agent access to infinite, persistent context.

Memory Types

Four distinct memory systems working together to give agents comprehensive understanding.

🕐
Episodic Memory
Event-based memory with timestamps. Tracks what happened, when, and by whom.
  • Timestamped events
  • Time range queries
  • Conversation linking
  • Who did what tracking
🧠
Knowledge Graph
Connected understanding of your codebase—relationships, dependencies, architecture.
  • Kuzu-based graph DB
  • Custom record types
  • Edge relationships
  • Cypher-like queries
🔍
Vector Database
Semantic search across all project knowledge. Find by meaning, not just keywords.
  • Configurable embeddings
  • Hybrid search
  • Metadata filtering
  • Collection namespaces
🗄️
Structured Memory
JSON and Markdown storage for specs, documentation, notes, and structured data.
  • Schema validation
  • Version tracking
  • Full-text search
  • Todo tracking

Knowledge Graph Capabilities

Define custom schemas, create relationships, and query your codebase as a connected graph.

Supported Data Types

STRING INT64 INT32 DOUBLE FLOAT BOOL DATE TIMESTAMP UUID LIST MAP

Graph Operations

  • ● Create/query records (nodes)
  • ● Create/query edges (relationships)
  • ● Pattern matching & traversal
  • ● Custom view templates

Why Persistent Memory Matters

No Re-Explaining
Agents remember context from weeks or months ago. No more "starting from scratch" every session.
Learning Standards
Agents learn your coding conventions, preferences, and patterns—and apply them consistently.
System Understanding
Knowledge graphs capture relationships. Agents understand your code as a system, not isolated files.
Semantic Discovery
Find relevant code by meaning. Ask "how do we handle auth?" and find all related code.

AI That Grows With Your Project

Build lasting institutional knowledge. Agents that truly understand your codebase.
