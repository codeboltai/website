# Codebolt Website Wireframe

This folder contains HTML wireframe pages for the Codebolt website - the world's first Agentic IDE for multi-agent development.

## Purpose

These wireframes are designed for **content flow finalization**. They are simple HTML pages focused on:
- Information architecture
- Content hierarchy
- User journey mapping
- Section organization

## Pages Included

### Core Pages
- `index.html` - Homepage with hero, problem/solution, features overview
- `features.html` - Detailed feature breakdown (swarms, agents, coordination, memory, etc.)
- `how-it-works.html` - Workflow explanations and getting started guide
- `use-cases.html` - Target audience personas and transformation stories
- `technology.html` - Technical architecture and stack details
- `pricing.html` - Pricing tiers and feature comparison
- `download.html` - Download options and quick start guide
- `about.html` - Company mission, vision, and team
- `enterprise.html` - Enterprise features, security, and contact form
- `docs.html` - Documentation hub and navigation

### Shared Assets
- `styles.css` - Shared wireframe styles

## Content Structure

The wireframes are based on the product documentation in `/wireframe/appdocs/feature/`:
- Application Overview
- Product Vision & Marketing
- Features and Functionality
- Value Proposition Matrix
- Technical Architecture
- Usage and Workflow Layer
- User Persona Journey

## Key Sections Covered

1. **Hero/Value Proposition** - Clear positioning as "Agentic IDE"
2. **Problem Statement** - Six key problems with current AI tools
3. **Solution Overview** - Four pillars (Swarms, Memory, Stigmergy, Observability)
4. **Features Deep Dive** - 150+ features organized by category
5. **How It Works** - Daily workflow and getting started
6. **Use Cases** - Enterprise, Startups, Solo, AI Engineers
7. **Technology** - Architecture, stack, memory systems
8. **Pricing** - Free, Pro, Team, Enterprise tiers
9. **Enterprise** - Security, compliance, ROI
