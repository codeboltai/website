Quick Start

Up and Running in 5 Minutes

1

Download & Install

Download the installer for your platform. Run the installer and follow the prompts. Launch Codebolt.

2

Configure LLM API Keys

Add your API keys for OpenAI, Anthropic, or other LLM providers. Codebolt supports multiple providers simultaneously.

3

Open Your Project

Open an existing codebase or create a new project. Codebolt will analyze your code structure automatically.

4

Create Your First Agent

Use a template from the Agent List Panel or create your own with the visual Agent Flow Creator.

5

Start Building

Open the Chat Panel, start your agent with a task, and watch it work. Review results and iterate.

Requirements

System Requirements

Minimum

  • 8 GB RAM
  • 4 CPU cores
  • 2 GB disk space
  • Internet connection
  • LLM API key (OpenAI, Anthropic, etc.)

Recommended

  • 16+ GB RAM
  • 8+ CPU cores
  • SSD with 10+ GB free
  • Fast internet connection
  • Multiple LLM API keys for redundancy
