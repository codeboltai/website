Three Paradigm Shifts

🐜

Stigmergy Coordination

Like ant colonies coordinate through pheromone trails, agents coordinate through environmental signals. No central orchestrator. No direct communication. Just agents reading and modifying a shared environment.

9 pheromone types across 5 temporal layers enable sophisticated coordination for 200+ agents without O(n²) overhead.

Externalized Memory

Memory is a feature of the environment, not the agent. When agents terminate, knowledge survives. The 3-layer Memoria architecture achieves 87% knowledge survival across generations.

5 memory layers: Persistent, Vector DB, Knowledge Graph, KV Store, Episodic.

Human Participation

You're a swarm member, not an observer. Humans use the same tools as agents: deposit pheromones, send mail, participate in deliberations, review changes.
