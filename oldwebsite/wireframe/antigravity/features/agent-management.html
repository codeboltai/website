Creation

Visual Agent Builder

Create agents without code using drag-and-drop flows. Or code them with full SDK access.

🎨 Visual Flow Design

Drag trigger, action, condition, and output nodes onto the canvas. Connect them to define agent behavior.

  • • Trigger: "On Job Assigned"
  • • Action: "Read File", "Code Review"
  • • Condition: "If Issues Found"
  • • Output: "Create PR", "Update Job"

🔧 Code-Based Agents

Full SDK access for custom logic in JavaScript/TypeScript. Agents have the same capabilities as humans.

  • • codebolt.fs.* — File system
  • • codebolt.jobs.* — Job management
  • • codebolt.memory.* — Memory ops
  • • codebolt.pheromone.* — Stigmergy
