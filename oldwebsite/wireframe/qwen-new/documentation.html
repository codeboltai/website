API Documentation

Agent Management API

POST
/api/v1/agents/create
Create a new agent with specified configuration and capabilities.
GET
/api/v1/agents/list
Retrieve a list of all active agents in the swarm.
PUT
/api/v1/agents/{id}/update
Update the configuration of an existing agent.
DELETE
/api/v1/agents/{id}/terminate
Gracefully terminate an agent and preserve its learnings.
