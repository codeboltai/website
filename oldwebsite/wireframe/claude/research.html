Featured Paper

Stigmergy-Based Coordination for Multi-Agent Software Engineering

First implementation of biological stigmergy principles in software development. Demonstrates how digital pheromone signals enable self-organizing agent coordination without central orchestrators.

Published: ACL 2026 • Authors: Research Team
Key Findings:
  • Zero coordination overhead at scale (100+ agents)
  • Emergent task decomposition without supervision
  • Linear performance scaling with agent count

Featured Paper

Externalized Memory Architecture for AI Agents

Novel approach to infinite context by moving memory from individual agents to the IDE level. Combines graph databases, vector search, and episodic memory for perfect project recall.

Published: NeurIPS 2025 • Authors: Research Team
Key Findings:
  • 100% recall accuracy across 6-month project history
  • Agent-to-agent knowledge transfer in <100ms
  • Semantic search outperforms RAG baselines by 40%
