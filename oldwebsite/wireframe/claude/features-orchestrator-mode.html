Orchestrator Mode

Full visibility and control when you need it. Orchestrator Mode gives you centralized oversight while maintaining the benefits of swarm intelligence.

Swarm Mode vs Orchestrator Mode

Codebolt offers two coordination modes. Use them together for optimal results.

🐜 Swarm Mode

Self-organizing agents coordinate via stigmergy. No central direction. Maximum autonomy. Best for:

  • Parallel feature development
  • Exploratory tasks
  • Well-defined work
  • Large agent counts (50+)

🎯 Orchestrator Mode

Centralized control with full visibility. You direct agent flow. Best for:

  • Critical path work
  • Complex dependencies
  • Learning/exploration
  • Human oversight required

Orchestrator Capabilities

Full Visibility Dashboard

Real-time view of all agents, their tasks, status, and outputs. Filter, search, and drill down into any agent's work.

Task Assignment

Manually assign tasks to specific agents. Override swarm decisions. Reallocate agents dynamically.

Intervention Controls

Pause any agent. Take over work. Redirect to new task. Resume when ready. Seamless handoff.

Approval Gates

Require human approval before agents proceed. Set gates for critical decisions, PR merges, or deployments.

Live Debugging

Inspect agent reasoning in real-time. View tool calls. See what they're thinking. Debug issues live.

Workflow Control

Define sequential workflows. Phase-based execution. Milestone tracking. Rollback capability.

When to Use Orchestrator Mode

  • Critical Security Work: Require approval for any security-related changes. Review all agent decisions manually.
  • Learning Onboarding: New developers watch orchestrator mode to learn how agents work. Safe exploration environment.
  • Complex Migrations: Orchestrator mode for multi-step migrations with precise ordering requirements. Swarm mode for independent subtasks.
  • Production Deployments: Orchestrator oversight for production releases. Swarm mode handles the bulk development work.
  • Debugging Issues: Switch to orchestrator mode to investigate bugs. Watch agents diagnose. Intervene when needed.

Control + Autonomy

Switch between modes as needed. Use swarm for scale, orchestrator for control.
