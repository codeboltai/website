# Codebolt Website Wireframes

This directory contains simple, content-focused HTML wireframes of all main pages from the Codebolt website. These wireframes strip away all design elements to focus purely on content structure and organization.

## Purpose

The wireframes are designed to help you:
- Quickly review all website content in one place
- Iterate on copy, messaging, and content hierarchy
- Plan content changes without worrying about design
- See how content flows across different pages

## How to Use

1. **Start with sitemap.html** - Open this file first to see all available wireframes
2. **Navigate freely** - Each page has a navigation bar to jump between main pages
3. **Review content** - Focus on copy, messaging, and content structure
4. **Make notes** - Document what needs to change
5. **Implement** - When ready, update the actual Next.js components

## Content Structure

Each wireframe follows a consistent pattern:
- **Section labels** - Gray badges indicating section types (Hero, Features, CTA, etc.)
- **Headlines** - Main page and section titles with hierarchy
- **Body text** - Descriptions and explanations
- **Metrics/stats** - Key numbers and value propositions
- **CTAs** - Call-to-action buttons with links
- **Navigation** - Links to other pages
- **Visual indicators** - Icons and symbols to represent features

## Design Philosophy

- **Minimal CSS** - Just enough for structure and readability
- **No responsive optimization** - Desktop-focused for content review
- **Black and white** - Neutral color scheme with blue (#0891b2) accents
- **System fonts** - Clean, simple typography
- **Content hierarchy** - Clear visual organization
- **Print-friendly** - Works well for offline review

## Quick Start
