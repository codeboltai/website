Memory Architecture

Externalized Memory Architecture BREAKTHROUGH

Memoria: The world's first complete externalized memory system for AI agents. Enable truly ephemeral agents with persistent collective intelligence across generations. First implementation of cross-generational knowledge transfer in artificial swarms.

Traditional LLMs have limited context windows—100-200K tokens. That's a keyhole view of your codebase. Codebolt solves this at the IDE/harness level through complete memory externalization. Memory lives outside the agent, giving every agent access to infinite, persistent context. Published Research: "Memoria: Externalized Memory Architecture for Ephemeral Multi-Agent Swarms with Persistent Collective Intelligence" (AAAI 2026).

Five-Layer Memory Architecture

Memoria implements five distinct memory systems working together to give agents comprehensive understanding and persistent knowledge across agent generations.

🕐
Episodic Memory
Event-based memory with timestamps. Tracks what happened, when, and by whom across agent generations.
  • Timestamped events across agent lifecycles
  • Time range queries spanning months
  • Conversation linking and context preservation
  • Historical decision tracking and rationale
🧠
Knowledge Graph
Connected understanding of your codebase—relationships, dependencies, architecture across agent generations.
  • Kuzu-based graph database with custom schemas
  • Entity-relationship queries across codebase
  • Pattern matching and traversal algorithms
  • Custom view templates for visualization
🔍
Vector Database
Semantic search across all project knowledge. Find by meaning, not just keywords, with persistent embeddings.
  • Configurable embedding models and dimensions
  • Hybrid search combining semantic and keyword
  • Metadata filtering and faceted search
  • Collection namespaces for organization
🗄️
Persistent Memory Layer
Structured long-term storage with exact-match retrieval for facts, learned patterns, and task outcomes.
  • PostgreSQL with JSON metadata storage
  • Complex filtering and temporal queries
  • Learned patterns and best practices
  • Task outcomes and performance metrics
