How to Use These Wireframes

1. Review Content: Open each HTML file in your browser to see the content structure

2. Iterate Quickly: Make notes on copy, messaging, and flow without worrying about design

3. Plan Changes: Identify what content needs updating, adding, or removing

4. Implement: When ready, update the actual Next.js components in the codebase

5. Navigation: Each page has a navigation bar to jump between main pages

Wireframe Design Notes

  • Minimal CSS for structure only - no design optimization
  • Black and white color scheme with blue (#0891b2) accents for CTAs
  • Simple typography using system fonts
  • Focus on readability and content hierarchy
  • Wireframe labels identify section types (Hero, Features, CTA, etc.)
  • Self-contained HTML files - no external dependencies
  • Print-friendly for offline review
