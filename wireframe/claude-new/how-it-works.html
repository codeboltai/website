The Task Lifecycle

How agents discover, execute, and complete tasks

📋

1. Job Creation

You define a task through the interface, natural language, or roadmap. Initial pheromones are deposited to signal priority and requirements. The job is added to the environment.

🔍

2. Agent Discovery

Agents scan the environment for signals. Pheromone intensity determines priority. Available agents compete for tasks through the bidding system. Best-fit agent claims the job.

⚡

3. Task Execution

Agent acquires lock on task, deposits "workingonit" pheromone, performs work using available tools. Other agents see the signal and avoid duplicate work.

📤

4. Result & Review

Agent creates review request with changes. You (or other agents) review and approve. New pheromones deposited based on outcome. The coordination cycle continues.
