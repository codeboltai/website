Core Concepts

Understanding the building blocks of Codebolt

Swarms and Agents

A swarm is a collection of AI agents working together. Each agent has specialized capabilities, permissions, and focus areas. Agents can be spawned dynamically based on workload and automatically terminated when no longer needed.

Jobs and Tasks

Jobs represent units of work—features, bugs, chores, or epics. Jobs can be decomposed into smaller sub-tasks. Agents claim jobs through a bidding system and lock them during execution to prevent conflicts.

Pheromones and Stigmergy

Pheromones are signals agents deposit on jobs to communicate state. Different types include importance, saturation, task_not_ready, and available. Stigmergy is the indirect coordination mechanism that enables self-organizing behavior.

Memory Systems

Codebolt includes four memory types: Episodic (what happened), JSON (structured data), Markdown (documentation), and Context (conversations). Plus a Knowledge Graph for codebase relationships.

Agent Development

Create custom agents for your specific workflows

🎨

Visual Agent Builder

Create agents by dragging and dropping nodes onto a canvas. Connect nodes to define workflow, set conditions for branching, and configure parameters. Export to code for advanced customization or deploy directly.

💻

Code-Based Agent SDK

For advanced users, create agents programmatically using our JavaScript/TypeScript SDK. Full access to all IDE functions including file operations, git commands, and memory management.

⚙️

Agent Configuration

Each agent has a `codeboltagent.yaml` configuration file defining capabilities, hooks, permissions, and LLM preferences. Use templates to get started quickly.

🔗

Hooks System

Define behavior using event-driven hooks. Pre/post execution hooks, custom handlers, and webhook integrations enable powerful automation workflows.

Integration

Connect Codebolt to your existing tools

MCP Support

Full Model Context Protocol (MCP) specification support. Install and manage MCP servers, register tools and resources, and ensure protocol compliance for extensibility.

CLI Access

Command-line interface for all operations. Agents can access CLI via SDK. Perfect for scripting and automation integration with your existing workflows.

WebSocket API

Real-time event streaming for agent coordination. Bidirectional communication enables live status updates and instant signal propagation.

REST API

HTTP endpoints for all features including CRUD operations, filtering, pagination, and authentication support. Build custom integrations on top of Codebolt.

SDK Quick Reference

Agents have full access equivalent to users

// Jobs API
await codebolt.jobs.create({
    name: "Fix authentication bug",
    type: "bug",
    priority: 4
});

// Memory API
await codebolt.memory.episodic.add({
    event: "bug_fixed",
    details: "Auth now works for all users"
});

// Calendar API
await codebolt.calendar.schedule({
    title: "Code review session",
    time: "2026-01-20T14:00:00Z",
    participants: ["agent-1", "agent-2"]
});

// Pheromone API
await codebolt.pheromone.deposit({
    jobId: "JOB-123",
    type: "importance",
    intensity: 8
});
