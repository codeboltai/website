How It Works

From Direction to Deployment
in Four Steps

Codebolt gives you the control of an orchestrator with the scale of a swarm. Set direction, let agents coordinate, review and approve.

Core Workflow
The Daily Development Cycle

Development Becomes
a 24/7 Operation

🎯

1. Set Direction

Define goals, requirements, and constraints. Create jobs in the Jobs Panel. Set priorities. Configure your swarm with the right roles and teams.

Tools: Jobs Panel, Roadmap Panel, Swarm Panel
💬

2. Agents Deliberate

Agents discuss approaches, vote on decisions, and plan execution. Deliberations create consensus without human bottleneck. Q&A sessions resolve ambiguities.

Tools: Deliberation Panel, Mail/Inbox
🐜

3. Pheromone Coordination

Agents self-organize around priorities through stigmergy. Important tasks attract attention. Saturated tasks repel. Blockers are signaled. No central orchestrator needed.

Tools: Pheromone System, Job Coordination
⚡

4. Swarm Executes

Agents work in parallel, 24/7. Each agent handles its assigned tasks. Code is written, tests are run, documentation is generated. Progress is tracked automatically.

Tools: Running Agents Panel, Code Panel, Terminal
✓

5. Human Review

Review agent-generated code. Approve, modify, or request changes. Merge approved work. Provide feedback that improves future agent performance.

Tools: Review Merge Request Panel, Agent Debug Panel

Deep Dive
Stigmergy-Based Coordination

How Agents Self-Organize
Without Central Control

Stigmergy is the coordination mechanism that allows ant colonies to build complex structures without a central commander. Codebolt brings this to software development.

[Diagram: Stigmergy Coordination Flow]

1. User deposits 'importance' pheromone on Job-X
2. Agent A senses high importance, claims Job-X (lock)
3. Agent A deposits 'saturation' pheromone
4. Agent B sees Job-X saturated, moves to Job-Y
5. Agent A finds Job-X complex, deposits 'request_split'
6. Agent C responds to split signal, proposes split
7. Deliberation: Agents vote on split proposal
8. Approved split creates Job-X1, Job-X2, Job-X3
9. New jobs get 'job_available' pheromone
10. Agents naturally distribute to new jobs

Key Benefits

No Single Point of Failure

Coordination continues even if individual agents fail. The system is inherently resilient.

Scales to Hundreds

Unlike orchestrator-based systems, stigmergy scales naturally to 100+ agents without bottlenecks.

Emergent Intelligence

Complex behaviors emerge from simple rules. The whole is greater than the sum of parts.
