FAQ

Frequently Asked Questions

Do I need to bring my own LLM API keys?

Yes, Codebolt is a platform that orchestrates AI agents. You provide your own API keys for OpenAI, Anthropic, or other LLM providers. This gives you full control over costs and model selection.

What counts as an "agent"?

An agent is a configured AI entity that can execute tasks. The limit refers to how many agents can be active in your swarm simultaneously. You can create unlimited agent configurations.

Can I try before I buy?

Absolutely. The Free tier is fully functional with up to 3 agents. Pro and Team plans include a 14-day free trial with full features.

Is my code secure?

Codebolt runs locally on your machine. Your code never leaves your infrastructure unless you explicitly configure external integrations. Enterprise customers can self-host for complete data sovereignty.

What's the difference between Codebolt and GitHub Copilot?

Copilot is a single-agent code completion tool. Codebolt is a multi-agent development platform. Copilot helps you write code faster. Codebolt lets you deploy swarms of agents that work in parallel, 24/7, with persistent memory and coordination.
