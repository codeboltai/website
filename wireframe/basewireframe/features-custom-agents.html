Agent Types

Four ways to create agents—from no-code to full programmatic control.

💻
Code Agents
Full programmatic control with the Codebolt SDK. Write custom agent logic in JavaScript/TypeScript.
  • Full SDK access to all IDE functions
  • Custom agent logic in JS/TypeScript
  • Hook system for event-driven behavior
  • Multi-LLM support within single agent
🔄
Flow Agents
Visual drag-and-drop agent creation. Design complex workflows by connecting nodes and configuring parameters.
  • Drag-and-drop interface
  • Visual node connections for control flow
  • Configure without code
  • Export to code for advanced customization
📋
Remix Agents
Start from proven templates and customize. Clone existing agents and modify for your needs.
  • Clone from template library
  • Customize for your workflows
  • Share across your organization
  • Version control and rollback
🌐
External Agents
Connect agents from external systems via WebSocket or HTTP. Integrate third-party tools and services.
  • WebSocket/HTTP endpoints
  • Join existing swarms
  • Participate in deliberations
  • Access pheromone signaling

Agent Hooks
