Changelog

What shipped, when.

2026-02-15 v0.9.0

Cross-environment reconciliation engine. Agents now coordinate across local, staging, and feature branches with automatic conflict resolution.

2026-02-01 v0.8.0

AI Playbooks — structured intelligence modules for encoding domain knowledge, constraints, and decision patterns. Composable and versionable.

2026-01-18 v0.7.0

Execution trace graph with full replay capability. Every agent decision is now traceable and auditable.

2026-01-05 v0.6.0

Drift engine v2 — two-level architecture with local Rust assessment and global JavaScript coordination. Tracks structural, scope, constraint, and coverage drift.

2025-12-20 v0.5.0

Stigmergy-based coordination replaces message-passing architecture. Significant scaling improvements for 10+ concurrent agents.

2025-12-01 v0.4.0

Narrative versioning with git tree-based snapshots. Agent attribution for every change — what changed, why, and by which agent.
